h1. Mole 

Mole is a simple time tracking tool for projects.

h2. Dependencies

* gem install sinatra activerecord
* gem install sqlite3-ruby # if you're using sqlite3

h2. Installing and Running

* git clone git://github.com/moomerman/mole.git
* cd mole
* rake mole:init
* ./mole.rb
* go to http://localhost:4567/
